## DESCRIPTION
The **goal** of this project is to create a **microservices** system using **Spring Boot 3** and **Java 17** . The following technologies and tools are used:
 * Spring Cloud
 * Docker
 * Swagger (OpenAPI)
 * Kafka
 * DDD (Domain Driven Design)
 * PostgreSQL
 * Postman
 * DBeaver
 * IntelliJ
 * Visual Studio Code
 * Sourcetree
 * Docker Desktop
